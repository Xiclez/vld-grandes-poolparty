El Evento

Después de grabar un set especial, Los Grandes y Viva La Dance se unen para celebrar su estreno como se debe: pool party, sonido fino y vibra nocturna que escala con cada hora. 🎶🌙

Un encuentro donde el techno, la energía colectiva y el espacio se alinean para crear una experiencia inmersiva — desde open decks hasta un cierre abierto que promete extender la noche. ⚡🖤

Prepárate para bailar dentro y fuera del agua, rodeado de luces, texturas y una curaduría musical pensada para llevarte de menos a más. 💃🕺💦

Llena Tus Datos

2
